Android + iPhone safe install flow

Установка зависит от платформы.

На Android Chrome кнопка появляется после события установки. На iPhone установка идет через Safari → Поделиться → На экран Домой.

Статус: ожидание проверки.

Как установить на iPhone

Открой сайт в Safari, нажми «Поделиться», затем выбери «На экран Домой» / Add to Home Screen.

Safari → Поделиться → На экран Домой
