# Astro Day PWA

Готовый набор файлов для GitHub Pages.

## Что загружать в репозиторий

- `index.html`
- `assets/manifest.webmanifest`
- `assets/icon.svg`
- `assets/sw.js`

## Как опубликовать

1. Создай новый public-репозиторий на GitHub.
2. Загрузить в него все файлы из этой папки с сохранением структуры.
3. Открой `Settings` -> `Pages`.
4. Включи `Deploy from a branch`.
5. Выбери ветку `main` и папку `/ (root)`.
6. Подожди появления ссылки вида `https://USERNAME.github.io/REPO/`.

## Как установить на iPhone

1. Открой опубликованную ссылку в Safari.
2. Нажми `Поделиться`.
3. Выбери `На экран Домой`.

## Структура

```text
index.html
assets/
  manifest.webmanifest
  icon.svg
  sw.js
```
